### 厨房尺寸
- 双开门冰箱 910

1.双开门冰箱尺寸一：冰箱的总容积301升以上，冷冻室的容积100升以上，一般尺寸在730*910*1780mm左右。

2.双开门冰箱尺寸二：冰箱总容积301升以上，冷冻室容积80-100升左右的，一般尺寸在920*820*1800mm。

3.双开门冰箱尺寸三：冰箱总容积600升左右，冷冻室容积100升左右的，一般尺寸升1770*920*740mm。

4. 双开门冰箱尺寸四：容量625/净容量556L 745*920*1775mm

5.双开门冰箱尺寸五：容量726/净容量661L1210*630*1852mm

6.双开门冰箱尺寸六：容量585/净容量534L925*816*1808mm

7.双开门冰箱尺寸七：容量605/净容量540L912*717*1798mm

8.双开门冰箱尺寸八：容量602/净容量520L795*890*1200mm

9.双开门冰箱尺寸九：净容量480L910*770*1755*mm

- 水槽 800 + 500
一般的双盆尺寸都在800mm左右

不锈钢厨房水槽的宽度一般都是450到500mm

台面安装水槽的时候，宽度需要预留至少100到150mm的空余。一般后三前七或者后7前八的规则。不过这也不是绝对，最好按实际情况咨询自己的设计师。

说到水槽的宽度问题，这里需要插播一句，因为水槽大多是安装在窗户位置，所以一定要了解清楚，你家厨房的窗户是内开窗还是啥。

如果是内开窗，就一定要考虑到水槽上的龙头位置，龙头千万别档着窗户的开和关。这是一件可以避免的事情，所以还是留意一下。

- 洗碗机
目前市场上的水槽洗碗机一般是分为三种规格：大型的洗碗机尺寸为600*850*600mm,中型的水槽洗碗机的尺寸为：450*800*600mm,小型的水槽洗碗机的尺寸为550*450*500mm。确切的尺寸要看具体的生产厂家，因为每一个生产厂洗碗机尺寸是不同的。例如家的洗碗机的尺寸是900*500*600mm


- 灶台
高度：灶台高度应在0.85m~0.85m之间或者0.93m~1.00m之间。

宽度：灶台宽度应在0.46m~0.6m之间或0.56m~0.63m之间。

- 消毒柜

一般来说长600mm左右,宽420-450mm左右，高650mm左右
